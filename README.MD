Tutorial for gmxplot – A GUI Plotter for GROMACS XVG Files
This tutorial provides a detailed, step‑by‑step guide to installing, configuring, and using gmxplot, a graphical application written in Python that lets you load, visualize, and analyse .xvg files generated by GROMACS molecular dynamics simulations.

1. What is gmxplot?
gmxplot is a cross‑platform desktop application that offers an intuitive point‑and‑click interface for plotting and analysing .xvg output files from GROMACS. It is designed to save you time and effort compared to using command‑line tools or external plotting software like Grace or QtiPlot.

With gmxplot you can:

Load many .xvg files at once from a chosen directory.

Visualise your data as line plots, bar charts, boxplots, or histograms.

Customise the look of your graphs with colour‑blind‑friendly palettes and scientific publication styles (via SciencePlots).

Annotate statistics – mean and standard deviation – directly on the plot.

Perform curve fitting (linear, quadratic, exponential) and display the fitted equation on the graph.

Export high‑resolution images (PNG, PDF, SVG) and the underlying data as CSV.

It is aimed at researchers and students who need quick, publication‑ready visualisations without learning complex scripting or leaving their desktop environment.

2. System Requirements
2.1 Python Version
gmxplot is written for Python 3.12.3 and works with Python 3.10 or newer.

2.2 Dependencies
The following Python libraries are required:

Library	Purpose
matplotlib	Core plotting engine
numpy	Numerical computations and array handling
pandas	Data manipulation and table structures
scipy	Curve fitting (curve_fit)
seaborn	Colour palettes and statistical plot aesthetics
scienceplots (optional)	Pre‑defined styles for scientific journals
tkinter is included with Python by default – no extra installation is needed.

3. Step‑by‑Step Installation
3.1 Create a Virtual Environment (Recommended)
bash
python3.12 -m venv gmxplot_env
source gmxplot_env/bin/activate   # On Linux/macOS
# or
gmxplot_env\Scripts\activate      # On Windows
3.2 Install Dependencies
You can install all required packages with a single command:

bash
pip install matplotlib numpy pandas scipy seaborn
If you want the scientific journal styles (strongly recommended), also install:

bash
pip install scienceplots
Note: If scienceplots causes compatibility issues with your version of Matplotlib, gmxplot will detect the problem and fall back to other available styles (seaborn‑v0_8, ggplot, etc.) without breaking.

3.3 Verify the Installation
bash
python3 -c "import matplotlib, numpy, pandas, scipy, seaborn; print('All good!')"
If you see no errors, you are ready to go.

4. Obtaining the Program
Save the complete Python script gmxplot.py (the final version we developed) in a folder of your choice. You can either copy the code from the previous answer or download it from a repository.

5. Running gmxplot
Open a terminal, navigate to the folder containing gmxplot.py, and run:

bash
python3 gmxplot.py
The main application window will appear.

6. User Interface Overview
The interface is divided into two main areas:

Left panel: all controls and options.

Right panel: the plotting area with an interactive toolbar (zoom, pan, save, etc.).

6.1 Key Controls
Control	Description
Working directory	The folder where your .xvg files are stored.
Browse...	Opens a file‑picker to choose a directory.
Load .xvg files	Scans the selected folder and lists all .xvg files.
File listbox	Shows found files. Click one to load and auto‑plot it.
Plot type	Select between Line, Bar, Boxplot, or Histogram.
Generate Plot	(Re)draw the plot with the current settings.
Style	Choose a visual style (e.g., default, science, ieee, nature) if SciencePlots is available.
Color palette	A selection of colour palettes – colorblind is recommended for accessibility.
Show legend	Toggle the legend on/off.
Annotate mean and std	Add annotations showing the mean and standard deviation of each Y‑series.
Curve fitting	Enable curve fitting.
Fit function	Choose Linear, Quadratic, or Exponential.
X column	Drop‑down to choose the column for the X‑axis.
Y columns	Multi‑select listbox for the Y‑axis series (Ctrl+click to select multiple).
Export Plot...	Save the current graph as PNG, PDF, or SVG.
Export Data (CSV)	Save the numerical data (the whole DataFrame) as a CSV file.
7. Step‑by‑Step Usage
7.1 Load Your Files
Click Browse... and navigate to the folder containing your .xvg files.

Click Load .xvg files.

All .xvg files in that folder (and its subfolders) will appear in the listbox.

7.2 Select a File
Simply click on any file in the list. The program will:

Parse the file.

Automatically detect the columns.

Update the column selection menus.

Generate a default plot (line plot with all Y columns) immediately.

7.3 Customise the Plot
You can adjust any of the following options and then click Generate Plot to refresh the view:

Plot type – switch to bar, boxplot, or histogram.

Columns – change the X column or select different Y columns (Ctrl+click).

Style – choose a publication‑ready style if SciencePlots is installed.

Palette – try colorblind for accessibility.

Legend – show or hide it.

Statistics – tick Annotate mean and std to display the values on the plot.

Curve fitting – activate it and select a fit function.

7.4 Export Your Results
Plot image: click Export Plot..., choose the format (PNG, PDF, SVG), and save. The resolution is set to 300 DPI.

Data: click Export Data (CSV) to save the complete data table as a CSV file.

8. Practical Example
Suppose you have a file rmsd.xvg with two columns: time (ps) and RMSD (nm).

Load the directory containing rmsd.xvg.

Click on rmsd.xvg – the line plot appears automatically.

Enable Curve fitting and choose Exponential – the program fits the curve and displays the equation.

For publication, set the style to nature and the palette to colorblind, then export as PDF.

9. Tips & Best Practices
Always use the colorblind palette to make your graphs accessible to all readers.

For scientific publications, choose styles like science or nature (requires SciencePlots).

If you have many files, you can select multiple entries; the program will load the first one, and you can switch by clicking on others.

Curve fitting is available only for line plots and uses the first Y column selected.

10. Troubleshooting Common Issues
10.1 AttributeError: module 'matplotlib.style' has no attribute 'core'
This happens when scienceplots is incompatible with your Matplotlib version. gmxplot already handles this gracefully: it will simply ignore scienceplots and use fallback styles. You can also uninstall scienceplots if you prefer:

bash
pip uninstall scienceplots
10.2 No Data Appears in the Plot
Verify that your .xvg file contains numeric data in the expected format.

Ensure you have selected at least one Y column.

Check that the X column contains numeric values.

10.3 The "Generate Plot" Button Is Not Visible
In the latest version, the button is placed right next to the plot type radio buttons to ensure it stays visible. If your window is too small, simply resize it.

10.4 Missing Dependencies
If any library is missing, install it with:

bash
pip install matplotlib numpy pandas scipy seaborn
11. Advanced Customisation
If you wish to extend or modify the program:

Add more fitting functions: update the FIT_FUNCTIONS dictionary in the code.

Change colour palettes: edit the palettes list inside create_widgets().

Adjust default resolution: change plt.rcParams['figure.dpi'] in the apply_style() method.
